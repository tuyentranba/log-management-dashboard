Import required modules at top of file: `import csv`, `import io`, `from datetime import datetime`, `import anyio`

- UTF-8 BOM ensures Excel recognizes UTF-8 encoding (without it, non-ASCII characters display as mojibake)
- StringIO buffer allows csv.writer to write to in-memory string (more efficient than string concatenation)
- Truncate/seek pattern prevents buffer from growing (only holds one row at a time)
- anyio.sleep(0) provides cancellation points for proper cleanup if client disconnects
- ISO 8601 timestamp format is machine-readable and sortable (per CONTEXT.md decision)
