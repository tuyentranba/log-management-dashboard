## Key Decisions

1. **Segmented control over dropdown**: Time range presets displayed as horizontal button group instead of dropdown. Better discoverability, shows all options at once, matches standard dashboard patterns.

2. **Inline custom date pickers**: When "Custom" selected, date pickers expand below button group instead of opening modal. Keeps context, matches Logs page pattern but visually integrated.

3. **8-point grid spacing**: All spacing values are multiples of 4px (Tailwind's standard):
   - Page padding: 24px (p-6)
   - Section gaps: 32px (space-y-8, gap-8)
   - Stat card gaps: 16px (gap-4)
   - Card internal padding: 24px (p-6 via shadcn defaults)

4. **URL state with nuqs**: Reused nuqs library from Logs page for consistent filter state management pattern. Enables shareable links with time range preserved.

5. **Removed Separator**: Replaced Separator component with space-y-8 for consistent visual rhythm. Simpler, cleaner, and more maintainable.

6. **Title hierarchy upgrade**: Upgraded page title from text-2xl (24px) to text-3xl (28px). Creates clearer hierarchy: page title (28px) > section titles (20px) > stat values (24px).
