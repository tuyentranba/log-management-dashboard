## Technical Implementation Details

### PUT /api/logs/{id} Endpoint
```python
@router.put("/logs/{log_id}", response_model=LogResponse)
async def update_log(
    log_id: int,
    log_data: LogCreate,
    db: AsyncSession = Depends(get_db)
):
    log = await db.get(Log, log_id)
    if not log:
        raise HTTPException(status_code=404, detail=f"Log with id {log_id} not found")

    log.timestamp = log_data.timestamp
    log.message = log_data.message
    log.severity = log_data.severity
    log.source = log_data.source

    await db.commit()
    await db.refresh(log)
    return log
```

**Key patterns:**
- Reuses LogCreate Pydantic schema for validation
- Full replacement (all fields updated)
- Returns SQLAlchemy model directly (Pydantic from_attributes handles conversion)
- 404 for non-existent logs, 400 for validation errors

### DELETE /api/logs/{id} Endpoint
```python
@router.delete("/logs/{log_id}", status_code=status.HTTP_204_NO_CONTENT)
async def delete_log(
    log_id: int,
    db: AsyncSession = Depends(get_db)
):
    log = await db.get(Log, log_id)
    if not log:
        raise HTTPException(status_code=404, detail=f"Log with id {log_id} not found")

    await db.delete(log)
    await db.commit()
    return None
```

**Key patterns:**
- Hard delete using `db.delete()` for permanent removal
- Returns 204 No Content (no response body)
- 404 for non-existent logs
